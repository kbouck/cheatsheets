





## Conversion

```java
String.valueOf(obj);              // Convert to string (Null-safe)
obj.toString();                   // Convert to string (NOT null-safe)
```





## Interpolation

```java
String.format("I have %s apples and %s bananas", apples, bananas);
String.format("I have %s fruit", apples + bananas)
```
